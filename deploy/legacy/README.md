# Legacy Deployment Files

This directory stores archived deployment assets that are no longer part of the canonical setup.

- `backend/` and `frontend/` contain the old container-oriented files.
- `deploy/` contains the old reverse-proxy and compose examples.
- `scripts/` contains archived shell entrypoints from earlier layouts.

Keep these files here only as historical reference. New changes should target the non-Docker local-process workflow documented in `docs/deployment.md`.
